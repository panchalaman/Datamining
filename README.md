# Data Mining

Projects and notebooks exploring data mining techniques and algorithms.

## Topics Covered

- **Classification**: Decision trees, Random Forest, SVM
- **Clustering**: K-Means, DBSCAN, Hierarchical Clustering
- **Association Rules**: Apriori algorithm, FP-Growth
- **Dimensionality Reduction**: PCA, t-SNE
- **Data Preprocessing**: Cleaning, normalization, feature engineering

## Tech Stack

- Python
- Jupyter Notebooks
- scikit-learn, Pandas, NumPy
- Matplotlib, Seaborn

## Structure

```
├── notebooks/       # Jupyter notebooks with analysis
├── data/            # Datasets used in projects
└── reports/         # Summary reports and findings
```

## Usage

```bash
git clone https://github.com/panchalaman/Datamining.git
cd Datamining
pip install -r requirements.txt
jupyter notebook
```
